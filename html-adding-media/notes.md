# Attributes for images
- src= : file path 
- alt= "[  media description ]"
- width= : size 
<br/>
Example: < img src=/home/rosexalba/html-lessons/html-semantics/imgCathedral.jpg alt= "picture of inside cathedral" width= "300" >
